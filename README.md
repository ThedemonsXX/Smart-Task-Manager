# 📋 Smart Task Manager

A responsive and interactive task management application built with **vanilla JavaScript**, focused on demonstrating core front-end development skills such as DOM manipulation, CRUD operations, and client-side data persistence.

---

## 🚀 Overview

Smart Task Manager allows users to efficiently create, edit, delete, and track tasks through a clean and intuitive interface. All data is stored locally in the browser using `localStorage`.

---

## ✨ Features

* Create tasks with title, description, and date
* Edit existing tasks via modal interface
* Delete tasks dynamically
* Track task status (Pending / Completed)
* Partial search functionality on task titles
* Persistent storage using `localStorage`
* Standardized date format (`YYYY-MM-DD HH:mm`)
* Real-time UI updates without page reload

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6+)
* LocalStorage API
